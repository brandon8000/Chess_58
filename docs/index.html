Just Another Chess App!

What This Is

My final project for my Software Methodology class (basically Intro to Software Engineering). The goal was to build upon an earlier project to implement a full Android application.

Features:

  • All rules implemented (except en passant)
  • Keeps a record of your moves and prints them at the end

Download

Make sure that you enable apps to be installed from unknown sources. A quick guide.
Open this website on your phone and click here to download. You should be prompted to install the application.

Stretch Goals

  • Save and reload games that are in progress
  • Save completed games in a format so that they can be viewed later
  • Add some artificial intelligence the player can play against
